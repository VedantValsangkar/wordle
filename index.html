Wordle

Wordle is a game of finding a hidden word.

Each guess must be a valid 5 letter word. Hit Enter button to submit.

After each guess, the color of blocks will change to show how close your guess was to hidden-word.

Green block represents letter in that block is at correct position in hidden-word.

Orange block represents letter in that block is in hidden-word but not in correct position.

Grey block represents letter in that block is not in hidden-word.
